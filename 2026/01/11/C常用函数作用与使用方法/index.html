C常用函数作用与使用方法

C语言常用标准库函数速查手册

本手册涵盖了 C 语言编程中最常用的头文件及其核心函数，适用于日常开发与复习。

1. 标准输入输出库 <stdio.h>

处理终端交互与文件读写。

函数 功能描述 示例
printf() 格式化输出数据到控制台 printf("Score: %d\n", 95);
scanf() 从控制台格式化输入数据 scanf("%d", &var);
getchar() 从输入流读取下一个字符 char c = getchar();
putchar() 输出一个字符到控制台 putchar('A');
fgets() 从流中读取一行字符串（安全，防止溢出） fgets(buf, sizeof(buf), stdin);
puts() 输出字符串并自动追加换行符 puts("Hello C");
fopen() 打开文件 FILE *f = fopen("test.txt", "r");
fclose() 关闭文件 fclose(f);

2. 字符串处理库 <string.h>

用于操作以 \0 结尾的字符数组。

函数 功能描述 备注
strlen() 计算字符串的实际长度 不计入结束符 \0
strcpy() 将源字符串复制到目标字符串 需确保目标空间足够
strncpy() 复制指定长度的字符串 比 strcpy 更安全
strcat() 将字符串追加到另一个字符串末尾 拼接操作
strcmp() 比较两个字符串的大小 相等返回 0
strstr() 在字符串中查找子串 返回子串首地址指针
memset() 将内存块填充为特定值 常用于清零数组
memcpy() 内存拷贝 用于任何类型的数据块

3. 标准工具库 <stdlib.h>

涵盖内存管理、随机数、转换及进程控制。

内存管理（堆内存）

  • malloc(size): 申请指定字节的内存，不初始化。
  • calloc(n, size): 申请并初始化为 0。
  • realloc(ptr, size): 重新调整已申请内存的大小。
  • free(ptr): 释放内存。务必与申请函数成对出现。

转换与工具

  • atoi(str): 字符串转整数（”123” -> 123）。
  • atof(str): 字符串转浮点数。
  • rand(): 返回一个伪随机数。
  • srand(seed): 设置随机数种子（常用 time(NULL) 作为种子）。
  • exit(status): 立即终止程序。

4. 数学库 <math.h>

进行复杂的算术运算。编译时通常需要链接数学库（-lm）。

  • pow(x, y): 计算 。
  • sqrt(x): 计算 。
  • abs(x): 整数绝对值（在 stdlib.h 中）。
  • fabs(x): 浮点数绝对值。
  • ceil(x): 向上取整（天板）。
  • floor(x): 向下取整（地板）。
  • **sin() / cos() / tan()**: 三角函数（参数为弧度）。

5. 字符分类库 <ctype.h>

用于判断字符的属性或进行简单的转换。

函数 判断条件
isalpha(c) 是否为字母
isdigit(c) 是否为数字
isalnum(c) 是否为字母或数字
isspace(c) 是否为空格、换行或制表符
isupper(c) 是否为大写字母
tolower(c) 转换为小写字母

6. 时间库 <time.h>

处理系统时间。

  • time(NULL): 获取当前系统的时间戳。
  • ctime(&t): 将时间戳转换为可读的字符串格式。
  • difftime(t1, t2): 计算两个时间点之间的秒数差。
